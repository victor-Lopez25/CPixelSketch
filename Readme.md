## CSketch

Sketch program made in C

## Dependencies

- [SDL2](https://www.libsdl.org/)

## Building

If on windows, use build.bat after initializing cl.exe (MSVC) with vcvarsall.bat

Warning: Must have sdl2.dll beside executable
