## CPixelSketch

Sketch program made in Odin with SDL2

WARNING: This is not complete yet!

## Dependencies

- SDL2

## Building

Windows:
```
build.bat
```

Linux:
```
build.sh
```

Must have SDL2.dll in the same directory as executable
